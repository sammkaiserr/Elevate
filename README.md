# Elevate

Elevate is a modern, social networking platform built for professionals and students. It allows users to connect, share insights, build their professional profiles, and discover opportunities through a sleek, responsive, and dynamic user interface.

## Key Features

- **Professional Profiles:** Create tailored profiles for Students, Professionals, and Mentors, highlighting skills and experiences.
- **Social Feed:** Share text and image posts, comment, upvote/downvote, and interact with the community.
- **Networking:** Send and accept connection requests to build a personalized professional network.
- **Real-Time Communication:** Direct messaging to connect seamlessly with other professionals (if integrated).
- **Search & Discovery:** Robust search functionally across posts, profiles, and domains.
- **Dark Mode:** Built-in theming logic for personalized viewing experiences.

## Tech Stack

- **Frontend:** React, Vite, Tailwind CSS / Custom CSS, React Router
- **Backend:** Node.js, Express, MongoDB
- **AI/Resume Backend:** Python
  - `groq` (LLM Inference)
  - `langchain` / `langchain-community` (AI Orchestration)
  - `PyPDF2` / `pdfplumber` (Resume Parsing)
  - `flask` / `fastapi` (API Routing)
- **Deployment:** Vercel

## Getting Started

### Prerequisites

- Node.js (v18+)
- npm or yarn

### Installation

1. Install dependencies:
   ```bash
   npm install
   ```
2. Start the development server:
   ```bash
   npm run dev
   ```
